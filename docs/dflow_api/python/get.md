## 返回值<a name="zh-cn_topic_0000001489189325_zh-cn_topic_0000001304385288_zh-cn_topic_0000001265240890_section51595365"></a>

## 异常处理<a name="zh-cn_topic_0000001489189325_zh-cn_topic_0000001304385288_zh-cn_topic_0000001265240890_section61705107"></a>

如果block为False或者get超时，会抛出queue.Empty异常；

如果队列出队过程中发生重部署或者进程退出，会抛出DfAbortException异常，表示函数执行中断；

如果get失败，会抛出DfException异常，表示UDF内部出队过程中发生异常。
