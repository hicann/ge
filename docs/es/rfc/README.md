# RFC 文档

本目录用于存放 ES 模块的设计提案（RFC - Request for Comments）文档。

## 什么是 RFC？

RFC 是用于讨论和记录重要设计决策的文档格式。当需要对 ES 模块进行重大变更、新功能添加或架构改进时，应提交 RFC 文档。

## RFC 流程

1. 创建 RFC 文档（建议使用编号，如 `rfc-001-feature-name.md`）
2. 提交 Pull Request 进行讨论
3. 根据反馈进行修改
4. 获得批准后合并

## 当前 RFC

暂无 RFC 文档。

## 如何提交 RFC

1. 创建新的 RFC 文档
2. 包含以下内容：
   - 概述和目标
   - 详细设计
   - 实现计划
   - 影响分析
   - 替代方案
3. 提交 PR 并等待审查
