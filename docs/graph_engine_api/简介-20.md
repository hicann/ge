# 简介<a name="ZH-CN_TOPIC_0000002519215179"></a>

表示ES API的C风格的输出类型；C用户构图调用esb\_funcs.h接口即可，C++用户调用EsTensorHolder的类的对外接口即可。

## 需要包含的头文件<a name="section710017525388"></a>

```
#include <es_c_tensor_holder.h>
```
