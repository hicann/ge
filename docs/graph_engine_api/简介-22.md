# 简介<a name="ZH-CN_TOPIC_0000002487055294"></a>

节点输出表达类，用于C++构图，封装了C的张量操作接口，该类提供了对节点输出的各种操作，包括设置dataype、format、shape等信息，以及设置私有属性，算术运算操作等能力。

## 需要包含的头文件<a name="section710017525388"></a>

```
#include "es_tensor_holder.h"
```
