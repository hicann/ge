## 功能说明<a name="section44282627"></a>

将传入的资源转移给EsCGraphBuilder进行管理。

## 函数原型<a name="section1831611148519"></a>

```
T *AddResource(std::unique_ptr<T> value, std::function<void(void*)> deleter = DefaultDeleter<T>)
```
