# 简介<a name="ZH-CN_TOPIC_0000002516818499"></a>

支持使用用户注册的外置allocator功能。

## 需要包含的头文件<a name="section78885814919"></a>

```
#include <ge/ge_allocator.h>
```

## Public成员函数<a name="section1173524710"></a>

```
Allocator() = default
virtual ~Allocator() = default
Allocator(const Allocator &) = delete
Allocator &operator=(const Allocator &) = delete
virtual MemBlock *Malloc(size_t size) = 0
virtual void Free(MemBlock *block) = 0
virtual MemBlock *MallocAdvise(size_t size, void *addr)
```
