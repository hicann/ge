# 简介<a name="ZH-CN_TOPIC_0000002487055296"></a>

C风格的图构建器结构体，C、C++的内部实现类，C用户构图调用esb\_funcs.h接口即可，C++用户调用EsGraphBuilder类的对外接口即可。

## 需要包含的头文件<a name="section710017525388"></a>

```
#include <es_c_graph_builder.h>
```
