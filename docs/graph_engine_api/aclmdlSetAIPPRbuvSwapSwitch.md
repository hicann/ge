## 功能说明<a name="section259105813316"></a>

动态AIPP场景下，设置是否交换R通道与B通道、或者是否交换U通道与V通道。

## 函数原型<a name="section2067518173415"></a>

```
aclError aclmdlSetAIPPRbuvSwapSwitch(aclmdlAIPP *aippParmsSet, int8_t rbuvSwapSwitch)
```
