## 功能说明<a name="section259105813316"></a>

动态AIPP场景下，必须设置原始图片的宽和高。

## 函数原型<a name="section2067518173415"></a>

```
aclError aclmdlSetAIPPSrcImageSize(aclmdlAIPP *aippParmsSet, int32_t srcImageSizeW, int32_t srcImageSizeH)
```
