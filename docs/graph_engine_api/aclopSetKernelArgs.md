## 功能说明<a name="section36583473819"></a>

动态Shape场景下，设置算子Tiling参数、执行并发数。

## 函数原型<a name="section13230182415108"></a>

```
aclError aclopSetKernelArgs(aclopKernelDesc *kernelDesc,
const char *kernelId,
uint32_t blockDim,
const void *args,
uint32_t argSize)
```
